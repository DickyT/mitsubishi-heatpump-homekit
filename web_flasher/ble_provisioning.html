Kiri Bridge
BLE Provisioning

Configure WiFi over Bluetooth from this browser. If you just received your Kiri Bridge, power it on first. When the status LED is rapidly flashing blue, the device is ready for this flow.

Browser Chrome or Edge Connection Web Bluetooth Security ESP-IDF Security0

Step 1 · Confirm Blue Flashing LED

Make sure your Kiri Bridge is powered on and the LED is rapidly flashing blue. That means installer mode or the production rescue provisioning window is active. If the LED is not flashing blue, restart the device or hold the provisioning button on production firmware until the BLE window opens.

Step 2 · Select Kiri Device

Select the BLE device named PROV_KIRI_.... The browser will keep this BLE session for the next steps.

Step 3 · Enter WiFi

Enter the WiFi credentials that should be written to the ESP32. Password is intentionally visible here so installers can verify it before sending.

Step 4 · Wait for Connection

The ESP32 is trying to join your WiFi. If this succeeds, the device should reboot or continue with the newly saved network credentials.
