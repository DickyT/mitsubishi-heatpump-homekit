New Kiri Bridge?

If you just received your Kiri Bridge and it already flashes blue after power-on, you probably do not need to flash firmware. Go straight to BLE WiFi provisioning.

Step 1 · Verify Hardware

Plug in the Kiri Bridge over USB and let the browser check the ESP32 bootloader at 115200 baud. This lightweight verification reads the chip type and MAC only. The port stays reserved afterward so the next steps cannot accidentally switch devices.

Step 2 · Choose Firmware / OTA

Select the Kiri firmware or OTA file you want to install. The browser checks the manifest, file list, sizes, and checksums before continuing.

Step 3 · Confirm and Install

Review the verified hardware and selected firmware or OTA bundle. Installation writes the bootloader, partition table, OTA metadata, and app image to this ESP32.
