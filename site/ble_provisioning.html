WiFi setup

Join the network.

Use Bluetooth once to put WiFi credentials on Kiri Bridge. When the LED is blinking blue, this page can find it.

Use Chrome or Edge Signal Blue blink WiFi 2.4 GHz

Now

Bluetooth: checking...
Device: not selected
WiFi: not configured

1. Look for blue

A fast blue blink means Kiri Bridge is advertising over BLE and ready for WiFi setup. If you do not see it, restart the device.

2. Select Kiri Bridge

Choose the BLE device named PROV_KIRI_.... Keep this tab open until WiFi setup finishes.

3. Send WiFi

Enter the WiFi network Kiri Bridge should join. The password stays visible so you can catch typos before sending.

4. Open the portal

After WiFi connects, use the shown portal address to finish setup and pair with Apple Home.
