1. Pick the board

Connect Kiri Bridge over USB. Verification reads the ESP32 chip and MAC address, then keeps that serial port reserved for the rest of the flow.

2. Choose the package

Select a published Kiri package. The browser checks its manifest, image list, sizes, and checksums before writing anything.

3. Write firmware

Confirm the board and package. When writing finishes, unplug/restart the device manually so Chrome does not have to handle a USB reset.
