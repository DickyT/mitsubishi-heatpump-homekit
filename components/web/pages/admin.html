Edit NVS Values

This is a dangerous operation. You are editing the device_cfg namespace; bad WiFi, HomeKit, CN105, or GPIO values can make the device go offline or stop controlling the AC. Canceling writes nothing.
High risk operation. Make sure you know what you are doing. If the device is working now, do not edit raw NVS values casually; a bad value can make the device unreachable or require USB recovery.
